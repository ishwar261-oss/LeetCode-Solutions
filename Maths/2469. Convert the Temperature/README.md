# 2469. Convert the Temperature

> **Problem Link:** https://leetcode.com/problems/convert-the-temperature/

## Problem Statement

Given a temperature in Celsius, return an array containing:

- Temperature in Kelvin
- Temperature in Fahrenheit

The conversion formulas are:

- Kelvin = Celsius + 273.15
- Fahrenheit = Celsius × 1.80 + 32.00

---

## Approach

Use the given mathematical formulas to convert the temperature.

Calculate the temperature in Kelvin and Fahrenheit separately, store both values in a vector, and return it.

---

## Algorithm

1. Read the Celsius temperature.
2. Calculate Kelvin.
3. Calculate Fahrenheit.
4. Store both values in a vector.
5. Return the vector.
